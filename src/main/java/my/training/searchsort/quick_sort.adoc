= 퀵 정렬 (Quick Sort)
:toc:
:toc-title: 목차
:sectnums:
퀵 정렬은 **분할 정복 알고리즘**을 기반으로 하는 매우 빠른 정렬 알고리즘입니다. 평균 시간 복잡도는 O(n log n)이며, 배열의 원소를 피벗(pivot)을 기준으로 두 개의 부분 배열로 분할하여 재귀적으로 정렬합니다.

== 알고리즘 설명
1. 배열의 마지막 요소를 피벗으로 선택합니다.
2. 피벗보다 작은 요소들은 왼쪽으로, 큰 요소들은 오른쪽으로 이동시킵니다.
3. 피벗의 왼쪽과 오른쪽 부분 배열에 대해 다시 동일한 방식으로 퀵 정렬을 재귀적으로 수행합니다.

퀵 정렬 알고리즘은 먼저 배열을 분할하는 `partition` 메소드를 통해 피벗을 설정하고, +
피벗을 기준으로 작은 값과 큰 값을 교체합니다. +
이후 피벗을 기준으로 배열을 재귀적으로 정렬합니다.
