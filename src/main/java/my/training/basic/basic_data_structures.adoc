== 소개
기본 자료구조와 알고리즘은 프로그래밍 문제를 해결하기 위한 가장 기본적인 도구입니다.

=== 회문(Palindrome) 확인
- **문제 설명**: 주어진 문자열이 거꾸로 읽어도 동일한지 확인하는 문제입니다.
- **접근 방식**: 문자열의 양 끝에서부터 하나씩 비교하여 동일한지 확인합니다.

== 4. 해시 테이블(Hash Table)

=== 두 수의 합 (Two Sum)
- **문제 설명**: 배열에서 두 수를 더해 특정 타겟 값을 만들 수 있는 인덱스를 찾는 문제입니다.
- **접근 방식**: 해시 테이블을 사용하여 각 숫자와 그 인덱스를 저장한 후, 타겟에서 현재 숫자를 뺀 값이 해시 테이블에 존재하는지 확인합니다.
