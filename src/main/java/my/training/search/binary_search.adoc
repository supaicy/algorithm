= 이진 탐색 (Binary Search)
:toc:
:toc-title: 목차
:sectnums:

이진 탐색은 **정렬된 배열**에서 탐색하고자 하는 값을 찾는 효율적인 알고리즘입니다. +
시간 복잡도는 O(log n)으로 매우 빠르며, 탐색 범위를 절반으로 줄여나가며 값을 찾습니다.

== 알고리즘 설명
이진 탐색은 배열의 중간 값을 선택하고, 이 값과 목표 값을 비교합니다. +
목표 값이 중간 값보다 크면 배열의 왼쪽 절반을 버리고, 작으면 오른쪽 절반을 버립니다. +
이를 반복하여 값을 찾습니다.

----
include::BinarySearch.java[lines=3..23]

이 코드에서, `left`와 `right`는 탐색할 배열의 경계를 나타냅니다. +
중간 값을 기준으로 탐색 범위를 좁혀 나가며, 목표 값이 중간 값과 같으면 해당 인덱스를 반환합니다. +
만약 목표 값이 없을 경우 `-1`을 반환하여 실패를 나타냅니다.
